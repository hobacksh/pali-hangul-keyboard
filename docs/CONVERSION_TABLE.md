# 빠알리어 로마자 → 한글 정밀 표기 변환 테이블

## 변환 예시

### 예시 1: Buddhānaṃ
**단계별 변환**:
1. `Bu` → `붓` (b=ㅂ, u=ㅜ)
2. `ddhā` → `다〭` + `᷇` (ddh=ㄷ + 아랫점, ā=ㅏ + 장음표)
3. `naṃ` → `낭〬` (n=ㄴ, a=ㅏ, ṃ=ㅇ + 윗점)

**최종 결과**: `붓다〭᷇낭〬`

### 예시 2: sabbaññutaññāṇassa
**단계별 변환**:
1. `sab` → `삽` (s=ㅅ, a=ㅏ, b=ㅂ)
2. `ba` → `반` (b=ㅂ, a=ㅏ, -n)
3. `ññu` → `뉴̫` (ñ=ㄴ + 아랫물결, u=ㅜ)
4. `ta` → `딴` (t=ㄷ, a=ㅏ, -n)
5. `ññā` → `냐̫᷇` (ñ=ㄴ+ㅑ + 아랫물결, ā=장음표)
6. `ṇas` → `나〭` (ṇ=ㄴ + 아랫점, a=ㅏ, s=ㅅ)
7. `sa` → `사` (s=ㅅ, a=ㅏ)

**최종 결과**: `삽반뉴̫딴냐̫᷇나〭사`

### 예시 3: catunnaṃ paccayānaṃ
**단계별 변환**:
1. `ca` → `짜` (c=ㅉ, a=ㅏ)
2. `tun` → `뚠` (t=ㄸ, u=ㅜ, n=ㄴ)
3. `naṃ` → `낭〬` (n=ㄴ, a=ㅏ, ṃ=ㅇ + 윗점)
4. `pac` → `빳` (p=ㅃ, a=ㅏ, c=ㅊ)
5. `ca` → `짜` (c=ㅉ, a=ㅏ)
6. `yā` → `야᷇` (y=ㅇ+ㅑ, ā=장음표)
7. `naṃ` → `낭〬` (n=ㄴ, a=ㅏ, ṃ=ㅇ + 윗점)

**최종 결과**: `짜뚠낭〬 빳짜야᷇낭〬`

## 변환 우선순위

### 1. 다중 문자 패턴 먼저
```
ddh → ㄷ (권설음)
kh → ㅋ
th → ㅌ
ph → ㅍ
```

### 2. 특수 문자 처리
```
ṭ, ṭh, ḍ, ḍh, ṇ → 아랫점 추가
ñ → 아랫물결 추가
ṅ → 윗점 추가
ṃ → 윗점 추가
ā, ī, ū → 장음표 추가
```

### 3. 단일 자음
```
k, g, c, j, t, d, n, p, b, m, y, r, l, v, s, h
```
