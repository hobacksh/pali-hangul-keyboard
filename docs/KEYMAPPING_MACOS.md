# PaliHangul 키보드 매핑 - macOS

## 개요

macOS용 빠알리 한글 입력기 (IME) 키 매핑 정의

**날짜**: 2026-02-12  
**버전**: 1.0  
**플랫폼**: macOS (InputMethodKit)

---

## 기본 입력

### 한글 입력
- 일반 한글 자판 사용
- 예: `ㄱ`, `ㄴ`, `ㄷ`, `ㄹ`, `ㅁ`, `ㅂ`, `ㅅ`, `ㅇ`, `ㅈ`, `ㅋ`, `ㅌ`, `ㅍ`, `ㅎ`

### 모음
- 예: `ㅏ`, `ㅑ`, `ㅓ`, `ㅕ`, `ㅗ`, `ㅛ`, `ㅜ`, `ㅠ`, `ㅡ`, `ㅣ`

---

## 결합 문자 (Combining Diacritics)

### 기본 4개 (필수)

| 단축키 | 결합 문자 | 유니코드 | 명칭 | 용도 |
|--------|----------|----------|------|------|
| `Option + D` | 〭 | U+302D | 아랫점 | 권설음 (ṭ, ṭh, ḍ, ḍh, ṇ, ḷ) |
| `Option + U` | 〬 | U+302C | 윗점 | 비음 (ṅ, ṃ) |
| `Option + M` | ᷇ | U+1DC7 | 장음표 | 장모음 (ā, ī, ū) |
| `Option + T` | ̫ | U+032B | 아랫물결 | 경구개 비음 (ñ) |

**입력 방식**:
1. 한글 입력 (예: `나`)
2. `Option + U` 누름
3. 결과: `나〬` (윗점이 붙음)

---

### 추가 결합 문자 (선택적)

| 단축키 | 결합 문자 | 유니코드 | 명칭 | 용도 |
|--------|----------|----------|------|------|
| `Option + L` | ͌ | U+034C | L 로마항소 | ㄹ 명시 (lomahaṁso) |
| `Option + V` | ᷃ | U+1DC3 | v 표시 | v 자음 구분 |
| `Option + Y` | ᷾ | U+1DFE | y 표시 | y 자음 구분 (miccheyya̿) |
| `Option + O` | ̿ | U+033F | 중복 표시 | 이중 자음 표시 |
| `Option + Shift + O` | ͂ | U+0342 | 대체 중복 | 중복 대체 (나눔체/끄레도체) |

---

### 위치 명시 (고급)

| 단축키 | 결합 문자 | 유니코드 | 명칭 | 용도 |
|--------|----------|----------|------|------|
| `Option + [` | ˕ | U+02D5 | 아랫점 위치 | 아랫점이 위의 자음임 명시 (수난뚜) |
| `Option + ]` | ˔ | U+02D4 | 윗점 위치 | 윗점이 위의 자음임 명시 (아밥바) |

## 구두점 (Private Use Area)

| 단축키 | 문자 | 유니코드 | 명칭 | 용도 |
|--------|------|----------|------|------|
| `Option + ,` | 󰠁 | U+0F0801 | 빠알리 쉼표 | 경전 구분 |
| `Option + .` | 󰠁 | U+0F0307 | 빠알리 마침표 | 경전 종결 |

**참고**: PUA (Private Use Area) 문자는 PaliHangul 폰트에서만 표시됨

## 입력 예시

### 예시 1: Buddhānaṃ (붓다낭)

**정밀 표기**: `붓〭다᷇낭〬`

**입력 순서**:
1. `ㅂ` + `ㅜ` + `ㅅ` → `붓`
2. `Option + D` → `붓〭` (아랫점)
3. `ㄷ` + `ㅏ` → `붓〭다`
4. `Option + M` → `붓〭다᷇` (장음표)
5. `ㄴ` + `ㅏ` + `ㅇ` → `붓〭다᷇낭`
6. `Option + U` → `붓〭다᷇낭〬` (윗점)

### 예시 2: sabbaññutaññāṇassa (삽반뉴딴냐낫사)

**정밀 표기**: `삽반뉴̫딴냐̫᷇나〭사`

**입력 순서**:
1. `삽반` 입력
2. `뉴` 입력
3. `Option + T` → `뉴̫` (아랫물결)
4. `딴` 입력
5. `냐` 입력
6. `Option + T` → `냐̫` (아랫물결)
7. `Option + M` → `냐̫᷇` (장음표)
8. `나` 입력
9. `Option + D` → `나〭` (아랫점)
10. `사` 입력

### 예시 3: saṅgha (상가)

**정밀 표기**: `상〬가`

**입력 순서**:
1. `ㅅ` + `ㅏ` + `ㅇ` → `상`
2. `Option + U` → `상〬` (윗점)
3. `ㄱ` + `ㅏ` → `상〬가`

- `D` = 아랫점 〭

## 시스템 요구사항

- **macOS**: 10.13 (High Sierra) 이상
- **폰트**: PaliHangul-Regular.otf 설치 필수
- **권한**: 입력 소스 추가 권한 필요

---

## 설치 방법

1. **폰트 설치**:
   ```bash
   open ~/PaliHangul-Regular.otf
   ```

2. **IME 설치** (향후):
   - `PaliHangul.app` 다운로드
   - `/Library/Input Methods/` 폴더에 복사
   - 시스템 환경설정 → 키보드 → 입력 소스 → PaliHangul 추가

3. **사용**:
   - 메뉴바에서 PaliHangul 선택
   - 한글 입력 + Option 키 조합
