## 빠알리어 음운 체계

### 자음 (33개)
빠알리어에는 다음과 같은 자음 체계가 있습니다:

#### 무성음 / 유성음 / 비음 / 반자음

| 조음 위치 | 무성 | 유성 | 비음 | 기타 |
|----------|------|------|------|------|
| 연구개 (Velar) | k, kh | g, gh | ṅ | |
| 경구개 (Palatal) | c, ch | j, jh | ñ | y |
| 권설 (Retroflex) | ṭ, ṭh | ḍ, ḍh | ṇ | r, ḷ |
| 치 (Dental) | t, th | d, dh | n | l |
| 순 (Labial) | p, ph | b, bh | m | v |

### 모음 (8개)

| 단모음 | 장모음 |
|--------|--------|
| a | ā |
| i | ī |
| u | ū |
| e | (장음 고정) |
| o | (장음 고정) |

## 한글 매핑 규칙 (초안)

### 자음 매핑

#### 무성음 (ka-계열)
- k → ㄱ (가, 각, 칵)
- kh → ㅋ (카, 칸, 칸)
- g → ㄱ (가, 각)
- gh → ㄱㅎ (가흐)

#### 비음
- ṅ → ㅇ (앙, 강의 ㅇ 받침)
- ñ → ㄴ + ㅑ (냐, 냔)
- ṇ → ㄴ (나, 난)
- n → ㄴ (나, 난)
- m → ㅁ (마, 맘)

### 모음 매핑

| 빠알리어 | 한글 | 예시 |
|----------|------|------|
| a | ㅏ | 가, 나, 마 |
| ā | ㅏㅏ or ㅏ+받침 | 까, 나아 |
| i | ㅣ | 기, 니, 미 |
| ī | ㅣㅣ or ㅣ+받침 | 끼, 니이 |
| u | ㅜ | 구, 누, 무 |
| ū | ㅜㅜ or ㅜ+받침 | 꾸, 누우 |
| e | ㅔ or ㅐ | 게, 네 |
| o | ㅗ | 고, 노, 모 |
