## 표기 철학

**"뿌라베다숫따 법문"의 표기법을 기반으로 하되,**  
**Romanized Pali 입력에 최적화된 단순화된 버전을 구현합니다.**

- ✅ 권설음 구별 (◌̣)
- ✅ 장모음 명시 (◌̄)
- ✅ 비음 구별 (ṅ/ñ/ṇ/n/m)
- ❌ 자음 중복 표시 (◌̎) - 불필요
- ❌ yi/vo 구별 (◌̊/◌̆) - 불필요

## 빠알리어 음운 체계 (출처: 뿌라베다숫따 법문)

### 자음 5단 분류

| 조음 위치 | 무성무기 | 무성유기 | 유성무기 | 유성유기 | 비음 | 기타 |
|----------|----------|----------|----------|----------|------|------|
| **후음** (Velar) | k 까 | kh 카 | g 가 | gh 가̎ | ṅ 앙 | |
| **구개음** (Palatal) | c 짜 | ch 차 | j 자 | jh 자̎ | ñ 냐 | y 야 |
| **권설음** (Retroflex) | ṭ 따̣ | ṭh 타̣ | ḍ 다̣ | ḍh 다̣̎ | ṇ 나̣ | r 라̣, ḷ 라̣ |
| **치음** (Dental) | t 따 | th 타 | d 다 | dh 다̎ | n 나 | l 라 |
| **순음** (Labial) | p 빠 | ph 파 | b 바 | bh 바̎ | m 마 | v 와 |

**기타 자음**:
- **마찰음**: s 사 (또는 싸)
- **기식음**: h 하
- **억제음**: ṁ 앙 (위치에 따라 변이)

### 모음 (8개)

| 종류 | Pali | 한글 표기 |
|------|------|-----------|
| **단모음** | a | 아 |
| | i | 이 |
| | u | 우 |
| **장모음** | ā | 아̄ (또는 아) |
| | ī | 이̄ (또는 이) |
| | ū | 우̄ (또는 우) |
| **복모음** | e | 에 |
| | o | 오 |

**주의**: 단모음도 겹자음 앞이나 억제음(ṁ) 앞에서는 장음으로 발음됨

## 한글 매핑 규칙 (확정)

### 자음 매핑 (5단 체계)

#### 후음 (Velar)
- k → 까 (각)
- kh → 카 (칸)
- g → 가 (간)
- gh → 가 + ◌̎ (가̎)
- ṅ → 앙 (강의 받침)

#### 구개음 (Palatal)
- c → 짜 (짝)
- ch → 차 (찬)
- j → 자 (잔)
- jh → 자 + ◌̎ (자̎)
- ñ → 냐 (냔)
- y → 야 (얀)

#### 권설음 (Retroflex) ⭐
- ṭ → 따 + ◌̣ (따̣)
- ṭh → 타 + ◌̣ (타̣)
- ḍ → 다 + ◌̣ (다̣)
- ḍh → 다 + ◌̣̎ (다̣̎)
- ṇ → 나 + ◌̣ (나̣)
- r → 라 (란)
- ḷ → 라 + ◌̣ (라̣)

#### 치음 (Dental)
- t → 따 (딴)
- th → 타 (탄)
- d → 다 (단)
- dh → 다 + ◌̎ (다̎)
- n → 나 (난)
- l → 라 (란)

#### 순음 (Labial)
- p → 빠 (빤)
- ph → 파 (판)
- b → 바 (반)
- bh → 바 + ◌̎ (바̎)
- m → 마 (만)
- v → 와 (완)

#### 기타 자음
- s → 사 (또는 싸)
- h → 하 (한)
- ṁ → 앙 (받침, 위치별 변이 가능)

### 모음 매핑

| Pali | 한글 표기 | 결합 문자 | 예시 |
|------|-----------|-----------|------|
| a | ㅏ | - | 가, 나, 마 |
| ā | ㅏ | + ◌̄ | 가̄, 나̄, 마̄ |
| i | ㅣ | - | 기, 니, 미 |
| ī | ㅣ | + ◌̄ | 기̄, 니̄, 미̄ |
| u | ㅜ | - | 구, 누, 무 |
| ū | ㅜ | + ◌̄ | 구̄, 누̄, 무̄ |
| e | ㅔ or ㅐ | - | 게, 네, 메 |
| o | ㅗ | - | 고, 노, 모 |

**중요**: 장모음은 **항상 마크론(◌̄)**으로 명시합니다.  
(원래 책은 겹자음/억제음 앞에서 생략하지만, 우리는 명확성을 위해 항상 표기)

## 결합 문자 (Combining Characters)

우리 프로젝트에서 사용하는 결합 문자는 **2종**입니다:

### 1. 아랫점 (Dot Below) - 권설음 표시
- **유니코드**: U+0323 (COMBINING DOT BELOW)
- **기능**: 권설음(Retroflex) 자음 구별
- **키 입력**: Option + D
- **예시**: 
  - `ṭ` → 따̣ (따 + ◌̣)
  - `ḍ` → 다̣ (다 + ◌̣)
  - `ṇ` → 나̣ (나 + ◌̣)

### 2. 마크론 (Macron) - 장모음 표시
- **유니코드**: U+0304 (COMBINING MACRON)
- **기능**: 장모음(Long vowel) 표시
- **키 입력**: Option + M
- **예시**:
  - `ā` → 가̄ (가 + ◌̄)
  - `ī` → 기̄ (기 + ◌̄)
  - `ū` → 구̄ (구 + ◌̄)

### ❌ 사용하지 않는 결합 문자

**"뿌라베다숫따 법문"에서는 사용하지만 우리는 생략**:
- ◌̎ (더블 어큐트) - 자음 중복 표시
- ◌̊ (어큐트) - 'y' + 모음 표시
- ◌̆ (캐론) - 'v' + 모음 표시

**이유**: Romanized Pali 입력 기준이므로 이미 `kamma`처럼 로마자로 중복 표기됨

## 권설음 vs 치음 비교 ⚠️

**중요**: 권설음과 치음은 **발음 위치가 다릅니다**!

| Pali | 한글 (권설음) | 한글 (치음) | 발음 차이 |
|------|---------------|-------------|-----------|
| ṭa | 따̣ | 따 | 혀를 입천장 앞쪽으로 말아올림 vs 윗잇몸에 부딪힘 |
| ṭha | 타̣ | 타 | 동일 |
| ḍa | 다̣ | 다 | 동일 |
| ḍha | 다̣̎ | 다̎ | 동일 |
| ṇa | 나̣ | 나 | 동일 |
| la (권설) | 라̣ | 라 | ḷ는 권설음 |

**아랫점(◌̣)이 있으면 권설음, 없으면 치음!**

## 억제음(ṁ) 발음 규칙

**Niggahita (ṁ)의 위치별 발음**:

| 다음 자음 | 발음 | 예시 |
|-----------|------|------|
| 후음 (k, g...) | 앙 | `saṅghaṁ` → 상감̄ |
| 구개음 (c, j...) | 안̃ | 구개비음 |
| 권설음 (ṭ, ḍ...) | 안̣ | 권설비음 |
| 치음 (t, d...) | 안 | 치비음 |
| 순음 (p, b...) | 암 | 순비음 |
| 모음/기타 | 암 or 앙 | 문맥 의존 |

**우리 프로젝트**: 일단 모두 **'앙'**으로 통일 (추후 정밀화 가능)

## 실제 예시

### Buddha (붓다)
- `b` → 바 (순음 유성무기)
- `u` → ㅜ (단모음)
- `ddh` → 다̎ (치음 유성유기, 받침 'ㄷ')
- `a` → ㅏ (단모음)
- **결과**: 붓다̎ (또는 붇다)

### Dhamma (담마)
- `dh` → 다̎ (치음 유성유기)
- `a` → ㅏ (단모음)
- `mm` → ㅁ (순음 비음, 받침)
- `a` → ㅏ (단모음)
- **결과**: 담마 (단모음이지만 겹자음 앞이므로 장음 발음)

### Saṅgha (상가)
- `s` → 사 (마찰음)
- `a` → ㅏ (단모음)
- `ṅ` → ㅇ (후음 비음, 받침)
- `gh` → 가̎ (후음 유성유기)
- `a` → ㅏ (단모음)
- **결과**: 상가̎

### Nibbāṇa (닙바나)
- `n` → 나 (치음 비음)
- `i` → ㅣ (단모음)
- `bb` → ㅂ (순음 무성무기, 겹자음)
- `ā` → ㅏ̄ (장모음!)
- `ṇ` → 나̣ (권설음 비음!)
- `a` → ㅏ (단모음)
- **결과**: 닙바̄나̣
