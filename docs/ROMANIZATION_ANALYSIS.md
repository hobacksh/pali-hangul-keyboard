# 빠알리어 로마자 ↔ 한글 대조 분석

## 실제 예시 (경전 텍스트)

### 1행
**로마자**: Nāññatra bojjhā tapasā,  
**한글**: 난냐뜨라 봇차 따빠사

**분석**:
- `Nā` → `난` (na + 장음)
- `ññ` → `냐` (ñ + ñ = 냐)
- `atra` → `뜨라` (a-tra)
- `bojjhā` → `봇차` (boj-jhā)
- `tapasā` → `따빠사` (ta-pa-sā)

### 2행
**로마자**: nāññatrindriyasaṃvara;  
**한글**: 난냐뜨린드리야상와라

**분석**:
- `nāñña` → `난냐` (nā + ñña)
- `tri` → `뜨리` (tri)
- `ndriya` → `드리야` (n-dri-ya)
- `saṃ` → `상` (saṃ = 상)
- `vara` → `와라` (va-ra)

### 3행
**로마자**: nāññatra sabbanissaggā,  
**한글**: 난냐뜨라 삽바닛삭가

**분석**:
- `nāñña` → `난냐`
- `tra` → `뜨라`
- `sabba` → `삽바` (sab-ba)
- `nissaggā` → `닛삭가` (nis-sag-gā)

### 4행
**로마자**: sotthiṃ passāmi pāṇinaṃ.  
**한글**: 솟팅 빳사미 빠니낭

**분석**:
- `sotthiṃ` → `솟팅` (sot-thiṃ)
- `passāmi` → `빳사미` (pas-sā-mi)
- `pāṇinaṃ` → `빠니낭` (pā-ṇi-naṃ)

---

## 핵심 발견

### 1. 장음 표시 없음
로마자에서 `ā, ī, ū`가 한글에서 **구분되지 않음**:
- `Nā` → `난` (장음표 없음)
- `tapasā` → `따빠사` (ā가 그냥 ㅏ)
- `passāmi` → `빳사미` (ā가 그냥 ㅏ)

### 2. 결합 문자 표시 없음
로마자의 특수 문자들이 한글에서 **단순화됨**:
- `ñ` (경구개 비음) → `냐` (그냥 ㄴ+ㅑ)
- `ṃ` (니깔라) → `ㅇ` 받침 (그냥 ㅇ)
- `ṇ` (권설 n) → `니` (그냥 ㄴ)

### 3. 이건 "간략 한글 표기"
이 텍스트는 **발음 기호가 생략된 일반 한글 표기**예요!

---

## 우리가 만들어야 할 것

### 옵션 1: 간략 표기 (이미지와 동일)
```
난냐뜨라 봇차 따빠사
```
→ 그냥 일반 한글 (우리 폰트 필요 없음)

### 옵션 2: 정밀 표기 (발음 기호 포함) ⭐
```
나〬난〬냐뜨라 봇〭차 따빠사᷇
```
→ 결합 문자로 정확한 발음 표시 (우리 폰트 필요!)

## 질문

JB님께서 원하시는 건:
1. **간략 표기** (이미지처럼, 일반 한글만)?
2. **정밀 표기** (발음 기호 포함, 우리 폰트 사용)?

아니면 **두 가지 모두** 지원? (변환 기능)

정밀 표기를 만들려면:
- 로마자 → 한글 (결합 문자 포함) 변환 테이블 필요
- IME에서 자동으로 결합 문자 추가
