## 핵심 패턴 발견

### 1. 장음 (ā, ī, ū) → 단순히 ㅏ, ㅣ, ㅜ
- `Buddhānaṃ` → `붓다낭` (ā = ㅏ)
- `jīvitassa` → `지위맛사` (ī = ㅣ)
- **장음 표시 전혀 없음**

### 2. 니깔라 (ṃ) → ㅇ 받침
- `naṃ` → `낭` (ṃ = ㅇ)
- `catunnaṃ` → `짜뚠낭` (ṃ = ㅇ)
- **윗점 표시 없음**

### 3. 경구개 비음 (ñ) → ㄴ + ㅑ
- `sabbaññutaññāṇassa` → `삽반뉴딴나낫사`
- `ñ` → `ㄴ` (아랫물결 표시 없음)

### 4. 권설음 (ṭ, ḍ, ṇ) → 일반 ㄷ, ㄴ
- `ṇassa` → `낫사` (ṇ = ㄴ, 아랫점 없음)
- **권설음 구분 전혀 없음**

### 5. 이중 자음 간략화
- `ddh` → `ㄷ` (하나로)
- `bb` → `ㅂ` (하나로)
- `cc` → `ㅉ` (하나로)

## 결론

### 이 텍스트는 "간략 한글 표기"
이 이미지의 한글은:
- ✅ 발음 위주 (듣기 쉽게)
- ❌ 발음 기호 없음 (장음, 윗점, 아랫점, 아랫물결 전부 생략)
- ❌ 권설음/경구개음 구분 없음

### 우리 폰트로 "정밀 한글 표기"를 만들면
```
로마자: Buddhānaṃ
간략:   붓다낭
정밀:   붓〭다᷇낭〬  (아랫점 + 장음표 + 윗점)
```

## 프로젝트 방향 결정 필요

### 옵션 A: 간략 표기만 (이미지와 동일)
- 로마자 → 일반 한글 변환
- **우리 폰트 불필요** (일반 한글로 충분)
- IME: 로마자 → 간략 한글 자동 변환

### 옵션 B: 정밀 표기 (발음 기호 포함) ⭐
- 로마자 → 한글 + 결합 문자
- **우리 폰트 필수**
- IME: 로마자 → 정밀 한글 자동 변환
- 예: `Buddhānaṃ` → `붓〭다᷇낭〬`

### 옵션 C: 간략 + 정밀 변환 지원
- 사용자가 선택 가능
- 두 가지 모두 지원
