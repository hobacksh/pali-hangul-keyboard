## 상세 설명

### 1. 장음 (U+FE20) - COMBINING LIGATURE LEFT HALF
- **모양**: 글자 위 가로 연결선 (왼쪽 반)
- **용도**: 장모음 ā, ī, ū 표시
- **예시**: 아︠ = ā, 이︠ = ī, 우︠ = ū

### 2 & 5. 캐론 / '와'의 표기 (U+030C) - COMBINING CARON
- **모양**: 위로 오목한 ˇ
- **용도**: o↔vo, u↔vu 구별
- **예시**: 오̌ = vo, 우̌ = vu
- `idamavoca` → 이다마오̌짜

### 3. 자음중복 (U+02BA) - MODIFIER LETTER DOUBLE PRIME
- **모양**: 위 겹따옴표 ʺ
- **용도**: 이중자음 구별 (ya↔yya 등)
- **예시**: 야ʺ = yya, 로ʺ = llo
- `miccheyya` → 밋체야ʺ
- `ullokeyyātha` → 울로ʺ께야ʺ타

### 4. '야'의 표기 (U+003C 기반) - 별도 형태 필요
- **모양**: 옆 꺽쇠 (<) - **글자 위에 작게 표시**
- **용도**: iṁ↔yiṁ 구별
- **예시**: `kappaṭṭhāyiṁ` → 잉 위에 < 표시
- ⚠️ **별도 글리프 디자인 필요** (일반 < 기호를 축소하여 위에 배치)

### 6. 유성대기음 (U+030A) - COMBINING RING ABOVE
- **모양**: 글자 위 빈 원 ○
- **용도**: 유성대기음 gha/jha/ḍha/dha/bha 표시
- **예시**: 가̊ = gha, 자̊ = jha, 다̊ = dha, 바̊ = bha

### 7. 권설음 (U+0325) - COMBINING RING BELOW
- **모양**: 글자 아래 빈 원 ○
- **용도**: 권설음 ṭ/ṭh/ḍ/ḍh/ṇ 표시
- **예시**: 따̥ = ṭa, 다̥ = ḍa, 나̥ = ṇa

### 8. L의 표기 (U+0303) - COMBINING TILDE
- **모양**: 위 물결 ~
- **용도**: l(설측음) 명시
- **예시**: 라̃ = la (설측음임을 명시)

### 9. 구개비음 (U+0330) - COMBINING TILDE BELOW
- **모양**: 아래 물결 ~
- **용도**: ñ 발음 표기
- **예시**: 나̰ = ña, 냐̰ = ñā

### 10. 쉼표 (U+05C0) - HEBREW PUNCTUATION PASEQ
- **모양**: 세로 선 ׀
- **용도**: 경전 텍스트의 구두점 (쉼표)

### 11. 마침표 (U+2225) - PARALLEL TO
- **모양**: 이중 세로 선 ∥
- **용도**: 경전 텍스트의 구두점 (마침표)
